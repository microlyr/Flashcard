# 📇 AI 智能闪卡 — 项目文档

## 当前状态

一个纯前端 HTML 工具（`flashcard.html`），拖入 Markdown 知识点文件，调用 AI（通义千问 / Claude / GPT）自动生成三种闪卡，在浏览器中背诵复习。

### 已实现功能

- [x] **AI 智能生成** — 从原始笔记自动生成挖空填空、问答题、选择题
- [x] **多 AI 支持** — 通义千问 Qwen（默认）、Anthropic Claude、OpenAI GPT、自定义兼容 API
- [x] **API Key 管理** — 浏览器 localStorage 存储，弹窗引导配置
- [x] **3D 翻转动效** — CSS 3D transform 翻转卡片
- [x] **三种卡片类型**
  - 🔤 挖空填空（Cloze）— 关键概念挖掉，翻转后高亮显示
  - ❓ 问答题（Q&A）— 思考后翻转看答案
  - 📋 选择题（Choice）— 4 选 1，点击选项自动翻转判对错
- [x] **掌握度追踪** — 每张卡片标记"我会了"/"没想起来"，进度条实时更新
- [x] **薄弱点复习** — 全部过完后可只复习标记为"没想起来"的卡片
- [x] **导出薄弱点** — 薄弱卡片导出为 .md 文件
- [x] **键盘快捷键** — Space 翻转 / ←→ 切换 / K 会了 / F 不会
- [x] **进度持久化** — localStorage 自动保存，刷新不丢
- [x] **响应式布局** — 手机/平板/桌面均可使用
- [x] **选项前缀防御** — 自动去除 AI 可能重复添加的 A.B.C.D. 前缀

### 技术栈

- 纯前端单文件 HTML + CSS + Vanilla JS
- AI API：Anthropic Messages API / OpenAI Chat Completions API（含 DashScope 兼容模式）
- 存储：`localStorage`
- 零依赖，无构建步骤

## 待完善事项

### 🔴 数据存储（高优先级）

> 目前所有数据仅存在 `localStorage` 中，局限性大。

- [ ] **历史记录存储**
  - 当前只有一份"当前进度"，没有历史复习记录
  - 需要记录：每次复习时间、每张卡片的掌握情况变化、复习轮次
  - 简单方案：使用 IndexedDB 或 Dexie.js 存储结构化历史数据
  - 需要设计数据模型：
    ```
    卡组 (deck): { id, name, createdAt, sourceFile, cardCount }
    卡片 (card): { id, deckId, type, front, back, options, answer }
    复习记录 (review): { id, cardId, deckId, result, timestamp, round }
    ```

- [ ] **多卡组管理**
  - 目前只能加载一个卡组，加载新文件会覆盖旧的
  - 需要：卡组列表、切换卡组、删除卡组、重命名

- [ ] **导入/导出卡组**
  - 导出：将当前卡组（含 AI 生成的卡片）导出为 JSON 文件，可分享给他人
  - 导入：导入之前导出的 JSON 卡组，无需重新调用 AI

### 🟡 智能复习算法（中优先级）

> 目前是简单的线性浏览 + 手动标记，缺乏科学的复习调度。

- [ ] **间隔重复（Spaced Repetition）**
  - 类似 Anki 的 SM-2 算法
  - 每张卡片根据掌握程度自动计算下次复习时间
  - 复习间隔：1 天 → 3 天 → 7 天 → 14 天 → 30 天
  - 标记"不会"的卡片缩短间隔，标记"会了"的卡片延长间隔

- [ ] **薄弱点智能判断**
  - 目前只有手动标记"没想起来"，判断维度单一
  - 应该综合判断：
    - 标记次数（连续 N 次标记"不会"= 薄弱）
    - 复习间隔（长时间未复习 = 可能遗忘）
    - 卡片类型（选择题蒙对的概率 > 填空题）
    - 响应时间（翻卡后思考时间过长 = 不熟练）
  - 薄弱点加权评分，动态调整复习优先级

- [ ] **每日复习提醒**
  - 到期卡片数量提醒
  - 可选：浏览器 Notification API 推送

### 🟢 用户体验优化（中优先级）

- [ ] **卡片编辑器**

## 快速开始

1. 用浏览器打开 `flashcard.html`
2. 去 [阿里云百炼](https://bailian.console.aliyun.com/#/api-key) 创建 API Key
3. 在弹窗中填入 Key（或点 ⚙️ 按钮手动配置）
4. 拖拽 `示例知识点.md`（或你自己的笔记）到页面
5. 等待 AI 生成卡片，开始背诵
